# GFL Assistant Lite

Ultra lightweight Girls Frontline helper app built with **ejs, expressjs, nodejs**.

Search for dolls and see info opened in new tabs/windows (enable popups please).

Hosted on [Heroku](https://gfl-lite.herokuapp.com).
